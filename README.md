# Sistema Puc-Commerce

Este projeto foi criado para o projeto final do curso de pós graduação "Desenvolvimento Web Full-Stack" da PUC-MINAS

## Para rodar localmente

### 1. Backend

```
$ cd server
$ npm install
$ npm start
```

### 2. Frontend

```
# Abra um novo terminal
$ cd client
$ npm install
$ npm start
```
